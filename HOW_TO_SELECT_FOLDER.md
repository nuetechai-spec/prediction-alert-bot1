## Troubleshooting

### "Folder not found"
- Make sure you're selecting the **inner** `prediction-alert-bot` folder
- Not the outer `prediction-alert-bot (1)` folder
- The correct path ends with: `...prediction-alert-bot (1)\prediction-alert-bot`

### "No files showing"
- Make sure you're in the right folder
- Check that files are actually there (use File Explorer)
- Try refreshing GitHub Desktop

### "Can't select folder"
- Make sure the folder exists
- Check you have permission to access it
- Try running GitHub Desktop as Administrator
