### Using GitHub Desktop (Easiest):
1. Select the entire `prediction-alert-bot` folder
2. GitHub Desktop automatically excludes files in `.gitignore`
3. It will skip `node_modules/`, `logs/`, `data/`, `.env` automatically

### Using Web Upload:
1. Select all files (Ctrl + A)
2. **Unselect** these folders:
   - `node_modules`
   - `logs`
   - `data`
3. Upload everything else
